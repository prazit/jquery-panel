::--- jQuery Panel or Appanel Styler Suite ---::

The jQuery Panel or Appanel Styler Suite is a collection of jQuery Plugin that used by Appanel Styler,
the collection has many functions listed below
- AlawaysShow: always show content when page scroll down  
- Glasses: show glasses behide dialog box or show glasses around element
- Sweep: change css class that work with the chains of animated class
